QR Code Generator

Remote MCP Server for Claude

MCP Endpoint: https://qrcode-mcp.apps.ramousse.com/api/mcp

Setup in Claude

  1. Open Settings
  2. Go to Connectors
  3. Click Add Custom Connector
  4. Paste the endpoint URL above
  5. Done!

Usage

Once connected, ask Claude to generate QR codes:

  1. "Generate a QR code for https://example.com"
  2. "Create a QR code with my email address"
  3. "Make a QR code for this WiFi network"

An interactive UI will appear in the conversation where you can customize and download your QR code.

Features

  1. Generate QR codes from any text, URL, phone number, or data
  2. Interactive UI with real-time preview
  3. Download QR codes as images
